Pangrama's Challenge

Jogo de digitação - Java - Swing

Motivação por atividades acadêmicas, este programa é um jogo de digitação de pangramas. O usuário deverá digitar corretamente o pangrama informado buscando sempre o menor tempo possível, enquanto poderá acompanhar seu resultado parcial em tempo real.

Ao concluir a frase, é informado o resultado e assim poderá ser consultado posteriormente em um scoreboard. É possível informar um nickname para facilitar a identificação do usuário que fez tal resultado.

Dentre algumas outras funcionalidades estão:

  • Alterar idioma dos pangramas
  • Selecionar pangrama para digitação
  • Alterar cor de background das teclas pressionadas
  • Funciona em sistemas operacionais Linux e Windows.

Para fazer o download basta acessar o repositório abaixo e clicar no botão de download:
https://github.com/allanalves23/aps_poo
Para sistemas Windows poderá ser necessário iniciar o programa como administrador para ser possível efetuar a gravação do nickname dos resultados.
